PROYECTOS

Argentina API

Argentina API es una API RESTful y GraphQL. Que podras utilizar para hacer practicas mientras aprendes datos de Argentina. Tendrás acceso a los datos de la Ciudad Capital, Coordenadas, Población, Superficie Km², Año de fundación, Fundador, Rango y Región Geográfica.

Construido con NodeJS y MongoDB.

Argentina API Site

El sitio oficial de Argentina API. Contiene todas las provincias y la documentación de como utilizar la API. Con un diseño llamativo y elegante.

Construido con puro HTML, CSS y JavaScript.
